# Sistema Integral de Gestión de Clientes, Servicios y Reservas

## Descripción general del proyecto

Este proyecto se basa en una solución a un problema propuesto en el curso Programación ofertado por la Universidad Nacional Abierta y A Distancia para los programas de Ingeniería Electrónica e Ingeniería en Sistema, el cual consta de desarrollar un sistema integral orientado a objetos sin el uso de bases de datos, para gestionar clientes, servicios y reservas para una empresa fisticia llamada Software FJ.

## Dependencias

En este proyecto se utilizaron ciertas bibliotecas intrinsecas de Python como lenguaje de programación principal, las cuales son:

* Tkinter.

## Requerimientos previos e instalación

En primer lugar se debe tener una versión de Python mayor o igual a 3, por ejemplo: la versión 3.

Luego se debe clonar y ubicarse sobre el repositorio de este proyecto con los siguientes comandos:

```bash
git clone [https://github.com/Jalbrin445/2026_proyecto_colaborativo_programacion.git]
cd [2026_proyecto_colaborativo_programacion]
```

## Ejecución del proyecto

```bash
python main.py
```
